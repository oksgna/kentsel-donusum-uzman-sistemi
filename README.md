Kentsel dönüşüm sürecinde binaların deprem riskini 
bina yılı, kat yüksekliği, beton kalitesi, kolonlardaki çatlaklar ve zemin sağlamlığı
üzerinden değerlendirerek risk puanı hesaplayan ve puana göre kentsel dönüşüm uygulanmalı veya risk yok gibi önerilerde bulunan bir sistemdir.

karar kriterleri:
Yapı yılı	:	< 2000, 2000–2018, >2018
Kat sayısı :		1–3, 4–7, 8+
Beton dayanım durumu	:	zayıf / orta / iyi
Kolon kiriş çatlak durumu	:	yok / hafif / belirgin
Zemin tipi	:	sağlam / orta / riskli

Kullanıcıdan aşağıdaki soruların cevapları alınır.
 'Bina kaç yılında yapıldı? '
 'Bina kaç katlı? '    
 'Beton sağlam mı(zayif/orta/iyi)?'
 'Kolonlardaki çatlakların sevise nedir?(belirgin/orta/hafif)'
 'Zemin sağlam mı?(riskli/orta/saglam)'

Bu karar kriterlerine göre gelen cevaplar üzerinden puanlama yapılır. Toplam puan'ın 3 ten küçük olması,3-6 arası veya 6'dan büyük olması gibi sonuçlara göre öneri yapılır.
'Yüksek risk: Kentsel dönüşüm gerekli
'Orta risk: Detaylı inceleme yapılmalı.'
'Düşük risk: Yapı güvenli.'
